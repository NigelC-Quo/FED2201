Coding Challenge #1 - Add Up

Went through this one with a for loop in mind but recursion could also work.

Just have a variable plus equal the iteration until it reaches one less than the number inputted, and then return the number plus the sum of that.

Coding Challenge #2 - Is it time for Milk and Cookies?

This one enlightened on how JS handles dates, I didn't know the months start from 0 to 11.

I found out that you couldn't just initialize a date object with that same date and then compare it because of object equality. You have to compare the exact copy of the Date object with that specific date and return it.

Coding Challenge #3 - Older than me

This one wasn't that difficult, since all you have to do is use the arguments defined in the constructor (name, age) and compare them with the class methods argument.

With each comparison of ages, adjust the sentences until it fits the criteria and then return them.
