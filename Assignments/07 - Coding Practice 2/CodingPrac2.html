Coding Challenge #6 - Matchstick Houses

This one was pretty straightforward. All I did was find the formula for the matchstick houses and then put in a condition in case it was 0. Also don't forget to put less than or equal to instead of just greater than.

Coding Challenge #7 - How much is True?

This one make you think on how to distinguish between variables that need to be matched and ones that do not. I searched for true statements in an if statement and made a count variable to show how many were in the array. I then made sure to keep the return statements out of the for loop so that it wouldn't prematurely exit.
